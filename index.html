User API

  • signup(POST): https://nodejsthreads.onrender.com/api/v1/user/signup
  • login(POST): https://nodejsthreads.onrender.com/api/v1/user/login
  • logout(POST): https://nodejsthreads.onrender.com/api/v1/user/logout

  • update(PUT): https://nodejsthreads.onrender.com/api/v1/user/update/userid
